# Discord.js v13 Music Bot
Music bot which using **Distube** package to play music

## Main Packages:
- distube v3.0.0-0
- ytdl-core v4.9.0
- ffmpeg-static v4.2.7
- opusscript v0.0.7
- @discordjs/voice v0.6.0
- @distube/soundcloud v0.2.2
